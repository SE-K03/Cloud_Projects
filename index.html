Secure Static Website Project

Project Overview

This project demonstrates the deployment of a secure static website using GitHub Pages. It highlights skills in cloud deployment, HTTPS security, version control, and adaptability using free tools.

Features

  • Static website hosted on GitHub Pages.
  • Secure access with HTTPS.
  • Version control via Git commits to track changes.
  • Accessible globally with GitHub Pages infrastructure.
  • Demonstrates adaptability by using free cloud tools instead of paid services like AWS.
